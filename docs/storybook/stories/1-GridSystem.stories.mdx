# Grid System

The following shows the classes for the grid-system in articles.
The column names are:

1. screen (+ inset)
2. page (+ inset)
3. body (+ outset + inset)
4. middle

The body has 6 equal columns, and the `middle` four columns can also be reffered to. The body `gutter`s are two columns on either side, and the `gutter-page` take one column and can be used with `middle` or `body-inset-left/right`.

These can be overridden by any theme, but genreally the layout should:

1. collapse the left margin first
2. collapse the right margin second
3. keep gutters to the sides in all views (including mobile)

When collapsed, the column content should take up the full body width (with the exception of body-gutters). The two `gutter-page` columns are also large enough that they should be full-body width.

When using in tailwind and `css` you need to import:

1. In your `app.css`, `@import '@myst-theme/styles/grid-system.css'`
2. In your tailwind config, `require('@myst-theme/styles').themeExtensions` (at least `gridTemplateColumns` and `gridColumn`)
3. (optional) Include the safe list of class names for all columns in your tailwind config: `require('@myst-theme/styles').safeList`
