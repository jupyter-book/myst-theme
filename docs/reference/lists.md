# Lists

## Unordered List

- First item
- Second item
- Third item
- Fourth item

## Nested Unordered

- First top-level item
  - Nested item 1
  - Nested item 2
    - Deeply nested
- Second top-level item

## Ordered List

1. First numbered item
2. Second numbered item
3. Third numbered item
4. Fourth numbered item
5. Fifth numbered item

## Nested Ordered

1. First item
   1. Nested 1
   2. Nested 2
2. Second item

## Task Lists

- [ ] Unchecked task
- [x] Checked task
- [ ] Parent task
  - [x] Completed subtask
  - [ ] Incomplete subtask

## Definition List

Term 1
: Definition of term 1

Term 2
: First definition of term 2
: Second definition of term 2

Term 3
: Definition of term 3 with more detail and explanation

## With Formatting

- Item with **bold** and *italic*
- Item with `code`
- Item with [link](https://mystmd.org)
